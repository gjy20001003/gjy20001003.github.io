常用工具

  • matsc：远程桌面管理工具
    步骤：目标机：控制面板——系统和安全——系统——远程设置——允许远程连接到此计算机。

           本机：cmd：matsc，填入ip地址和账户凭据

    请注意选择用户中可能存在用户组everyone，这个用户组致使所有人都可以连接到你的主机，按需设置可连接用户或用户组

  • regdit：注册表

  • gpedit.msc：组策略

    其中本地策略、账户策略比较重要，路径为：计算机配置—-windows设置—-安全设置—-本地策略\账户策略
    账户策略：
    密码策略中是密码复杂度、长度等要求，账户锁定策略为登陆失败处置，kerberos策略为域内凭证时间等策略

    本地策略：
    本地策略中审核策略是对各个操作行为的审核的开关，也是溯源和跟踪的重要依据，按需开启
    本地策略中的用户权限分配需按照最小权限原则进行处理
    安全选项则是对于域、服务器、账户、系统、网络等协议或者功能的安全性设置

gpmc.msc：域策略

wf.msc：防火墙设置

日志备份

创新一个空文件，可以看到属性是A，模拟备份ttt文件后，A标记被去除表示文件已备份，修改文件后重新出现A标记

日志备份原理：遍历需要备份的文件，查看文件属性是否是A，是A则进行备份后完成将属性转化成N，即进行差异备份

判断操作系统

  • 修改url参数，比如改成大写，正常的是windows，不正常的是linux
  • 使用ping命令，linux返回的TTL值一般比windows的值小一些，一般以100为划分线

过滤规则

操作 代码
过滤ip 源ip：ip.src eq ip
目的ip：ip.dst eq ip
过滤端口 tcp.port eq 80//源和目标都显示
tcp.dstport == ip 或 tcp.srcport == ip
tcp.port >= 1 and tcp.port <= 80
过滤协议 输入协议名，如tcp、udp、arp、icmp等
排除协议：如排除arp包，如!arp 或 not arp

http contains “HTTP/1.”
